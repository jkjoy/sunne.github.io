linux远程拷贝命令scp及出现not a regular file 解决

发表于2019-04-11|更新于2023-04-03|阅读量:

最近开始折腾ubuntu使用命令行上传下载还是挺方便的。 想

备份数据到本地 那就使用scp命令 scp提供了几个选项 在scp后加就行了

将远程主机192.168.1.23的/home/adm/目录下所有文件下载至本地/home/sun，则命令为：

scp -r root@192.168.1.23:/home/adm/ /home/sun/

回车后输入密码就可以了
ps linux 下scp传文件时错误
scp: /usr/tools: not a regular file 不能成功传送

1：有可能没权限 chmod 777

2: 在使用scp时加上-r 参数
